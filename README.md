## Features
- User Registration & Login (Customer/Admin)
- Vehicle Management (Add, Edit, Delete)
- Booking Management (Create, List, Update)
- Role-based Access Control
- JWT Authentication

## Technology Stack
- Backend: Node.js, Express.js, TypeScript
- Database: PostgreSQL
- ORM / DB Access: pg / node-postgres
- Authentication: JWT
- Deployment: Render
